## Welcome to the Tremor API

The Tremor API provides programmatic access to prediction market data from Polymarket and Kalshi. Query historical snapshots, track market movements, and build custom analytics with our REST API.

<Card
  title="Get Your API Key"
  icon="key"
  href="https://tremor.sh/settings"
>
  Generate an API key from your Tremor dashboard
</Card>

## Base URL

All API requests should be made to:

```
https://api.tremor.sh
```

## Authentication

All API endpoints require authentication using Bearer tokens. Include your API key in the `Authorization` header:

```bash
curl -X POST https://api.tremor.sh/query \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "SELECT * FROM polymarket_events LIMIT 10"}'
```

<Warning>
Keep your API key secure! Never commit it to version control or share it publicly.
</Warning>

## Rate Limits

<Info>
There are currently **no rate limits** on API requests. Query as frequently as you need.
</Info>

## Response Format

All API responses are returned in JSON format:

```json
{
  "success": true,
  "data": [...],
  "execution_time_ms": 156
}
```

## Error Handling

API errors return standard HTTP status codes with descriptive error messages:

<ResponseField name="400" type="Bad Request">
  Invalid query syntax or parameters
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Missing or invalid API key
</ResponseField>

<ResponseField name="500" type="Internal Server Error">
  Server error - please retry or contact support
</ResponseField>

## Quick Example

Here's a complete example of querying the top 10 markets by volume:

<CodeGroup>

```python Python
import requests

response = requests.post(
    "https://api.tremor.sh/query",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    },
    json={
        "query": """
            SELECT title, yes_probability, volume_24hr
            FROM polymarket_events
            WHERE active = true
            ORDER BY volume_24hr DESC
            LIMIT 10
        """
    }
)

data = response.json()
for market in data['data']:
    print(f"{market['title']}: ${market['volume_24hr']:,.2f}")
```

```javascript JavaScript
const response = await fetch('https://api.tremor.sh/query', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    query: `
      SELECT title, yes_probability, volume_24hr
      FROM polymarket_events
      WHERE active = true
      ORDER BY volume_24hr DESC
      LIMIT 10
    `
  })
});

const data = await response.json();
data.data.forEach(market => {
  console.log(`${market.title}: $${market.volume_24hr.toLocaleString()}`);
});
```

```bash cURL
curl -X POST https://api.tremor.sh/query \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "SELECT title, yes_probability, volume_24hr FROM polymarket_events WHERE active = true ORDER BY volume_24hr DESC LIMIT 10"
  }'
```

</CodeGroup>
