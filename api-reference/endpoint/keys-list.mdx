---
title: 'List API Keys'
openapi: 'GET /api/keys/list'
---

Return all active API keys for the authenticated user, including creation timestamps and the most recent usage time.

<Info>
API key management endpoints are rate limited to <strong>30 requests per minute</strong> per authenticated user.
</Info>

<Info>
Keys are filtered by <code>user_id</code> server-side; the endpoint never exposes other users' credentials.
</Info>
