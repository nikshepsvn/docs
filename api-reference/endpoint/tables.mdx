---
title: 'List Tables'
openapi: 'GET /api/tables'
---

Discover the datasets available in Tremor, including historical and latest snapshot tables for Polymarket and Kalshi.

<Info>
Table metadata endpoints are rate limited to <strong>60 requests per minute</strong> per API key.
</Info>

### Query parameters

<ParamField query="with_stats" type="boolean" default="false">
  Include approximate row counts and column metadata for each table.
</ParamField>

<ParamField query="refresh" type="boolean" default="false">
  Bypass the metadata cache and fetch fresh stats from ClickHouse.
</ParamField>

### Example Response

```json
{
  "tables": [
    {
      "namespace": "polymarket",
      "table": "events",
      "full_name": "polymarket.events",
      "source": "polymarket",
      "row_count": 8456782,
      "columns": [
        {"name": "id", "type": "String"},
        {"name": "title", "type": "String"},
        {"name": "yes_probability", "type": "Float64"},
        {"name": "sync_timestamp", "type": "DateTime64(3)"}
      ]
    },
    {
      "namespace": "kalshi",
      "table": "events",
      "full_name": "kalshi.events",
      "source": "kalshi",

        {"name": "id", "type": "String"},

}
```
