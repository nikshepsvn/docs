---
title: 'Table Schema'
openapi: 'GET /api/table/{namespace}/{table}'
---

Return column names and data types for any dataset. Use this endpoint to power schema browsers and to generate strongly typed query builders.

<Info>
Table metadata endpoints are rate limited to <strong>60 requests per minute</strong> per API key.
</Info>

<ParamField path="namespace" type="string" required>
  Data source namespace, such as <code>polymarket</code>, <code>kalshi</code>, or <code>internal</code>.
</ParamField>

<ParamField path="table" type="string" required>
  Table name within the namespace (for example <code>events</code>, <code>markets_latest</code>, or <code>sync_run_history</code>).
</ParamField>

### Example Response

```json
{
  "namespace": "polymarket",
  "table": "events",
  "columns": [
    {"name": "id", "type": "String"},
    {"name": "title", "type": "String"},
    {"name": "yes_probability", "type": "Float64"},
    {"name": "no_probability", "type": "Float64"},
    {"name": "volume_24hr", "type": "Float64"},
    {"name": "liquidity", "type": "Float64"},
    {"name": "active", "type": "Bool"},
    {"name": "sync_timestamp", "type": "DateTime64(3)"}
  ]
}
```

<Tip>
Combine this endpoint with <a href="/api-reference/endpoint/table-stats">column statistics</a> to surface min/max values, distinct counts, and null rates directly in your tools.
</Tip>
