## What is Tremor?

Tremor provides fast, reliable access to prediction market data from Polymarket and Kalshi. Built for traders, researchers, and enterprises who need historical market data and real-time analytics.

**Tech Stack:** React + Vite frontend, FastAPI backend, ClickHouse database, Claude AI assistant.

## Quick Example Queries

### Find Arbitrage Opportunities

```sql
SELECT
  p.title,
  p.yes_probability as polymarket_yes,
  k.yes_probability as kalshi_yes,
  ABS(p.yes_probability - k.yes_probability) as spread
FROM polymarket_events p
INNER JOIN kalshi_events k ON similarity(p.title, k.title) > 0.8
WHERE spread > 0.05
ORDER BY spread DESC;
```

### Track Market History

```sql
SELECT
  sync_timestamp,
  yes_probability,
  volume_24hr
FROM polymarket_events
WHERE id = 'trump-2024-election'
ORDER BY sync_timestamp DESC
LIMIT 1000;
```

### Volume Leaders

```sql
SELECT
  title,
  volume_24hr,
  liquidity
FROM polymarket_events
WHERE active = true
ORDER BY volume_24hr DESC
LIMIT 10;
```
